## Installation

Installation is done in three simple steps:

#### Clone the repo

```console
$ git clone https://github.com/putaindecode/ed-209
```

#### Run the installation

```console
$ npm install
```

#### Edit the config file

```console
$ cp config.example.json
$ $EDITOR config.json
```

## Usage

Start the bot:

```console
$ npm start
```
